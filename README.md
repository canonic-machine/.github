## The Stack

```
┌─────────────────────────────────────┐
│  LLM (Executor)                     │
├─────────────────────────────────────┤
│  CANONIC (Process)                  │
│  - Axioms                           │
│  - Validators (VaaS)                │
│  - Evidence (TaaS)                  │
├─────────────────────────────────────┤
│  Git (State Machine)                │
│  - Commits = State transitions      │
│  - Clones = Distributed ledger      │
│  - Encryption = Privacy layer       │

```

No databases. No servers. No cron. Just Git + CANONIC + LLM.
